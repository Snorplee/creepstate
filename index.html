üìö Usage Guide

üìÖ Interactive Timeline Navigation

  • Zoom: Use mouse wheel or zoom controls for different time scales (Hour/Day/Week/Month/Year/Decade)
  • Search: Type keywords to filter events in real-time
  • Categories: Filter by event types (Events, Reports, Legal, Investigations, Deaths, Putin connections)
  • Dark Mode: Toggle between light and dark themes with persistence
  • Expand Events: Click on event bubbles to see full details and evidence

üé¨ Slideshow Controls

  • Navigation: Arrow keys, spacebar, or touch/swipe gestures
  • Auto-Advance: Adjustable interval from 2-15 seconds
  • Keyboard Shortcuts: 'A' for auto-toggle, 'H' for help
  • Full Screen: F11 or F key for full screen presentation mode

üåê Network Visualization

  • Drag Nodes: Click and drag to reposition network elements
  • Filter Types: Show/hide people, organizations, locations, events
  • Physics Simulation: Toggle realistic node behavior and collision detection
  • Export: Save network visualizations as PNG images
